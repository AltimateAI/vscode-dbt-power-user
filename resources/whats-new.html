Highlights from the last few months. Toggle the checkbox below if you'd rather see this only when you open it manually.

🔧 Develop

  • CTE Profiler — see per-CTE timing and row counts as inline decorations while a query runs, so you can spot expensive sub-queries without rewriting them. (#1863)
  • YAML schema codelenses — Run/Test actions and model info hovers now appear directly on YAML schema files, not just on SQL. (#1872)
  • RisingWave adapter support — first-class detection for projects using the RisingWave adapter. (#1889)
  • Macro go-to-definition — resolves macros prefixed with their own project name (e.g. my_project.my_macro()). (#1881)

🧪 Test

  • Singular data tests — the “Run Test” action now recognises singular data tests, not just generic schema tests. (#1879)
  • Docs editor schema fix — stops writing both tests: and data_tests: for the same column, keeping schema YAML clean. (#1899)

🤝 Collaborate & AI

  • Altimate Code product hook — Query Results welcome tab now points to Altimate Code chat for context-aware help. (#1907, #1909)
  • Troubleshoot with Altimate — when a query fails, jump straight from the error into a guided troubleshooting flow. (#1906)
  • MCP tool schemas — schemas now strip $schema so Gemini and other strict consumers accept them. (#1891)

🛠️ Reliability & internals

  • Telemetry crash-paths hardened against undefined uri.fsPath / range.start, fixing several Sentry reports. (#1890)
  • Azure telemetry restored by passing the connection string through to TelemetryReporter. (#1912)
  • Build system migrated from webpack to rsbuild — faster compiles in development and CI. (#1762)
